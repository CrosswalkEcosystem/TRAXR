# TRAXR Roadmap

## Phase 0 – MVP (Completed)
- Static pool ingestion (JSON)
- Private scoring engine integration
- UI components (nodes, warnings, charts)

## Phase 1 – Indexer Foundation
- XRPL AMM scanning without XRPSCAN
- Trustline + issuer metadata collection
- Data normalization

## Phase 2 – Scoring Model v1.5
- Volatility metrics
- Issuer trust models
- Fee normalization improvements

## Phase 3 – API Expansion
- Full pool catalog
- Issuer analytics
- Alerts and anomaly detection

## Phase 4 – Production Indexer
- Full-history XRPL ingestion
- Live scoring
- Ecosystem partner integrations
